# Geth Docker

## For Production

### docker-compose

```console
$ export GETH_DATADIR=/data/geth
$ docker-compose up -d
```

### Kubernetes

Change [storage class](k8s/storageclass.yaml) and [pvc](k8s/pvc.yaml) first.

Start geth node

```
kubectl apply -f k8s
```
